Official Recruitment Website Redesign Project @ E.SUN FHC

Overview

This project aimed to improve recruiting process and experience to attract more potential talents to company. Thus, our team reconstructed the user flow and redesigned user interface of Official Recruitment Website to make it looks vibrant and easy-to-use.

My Role

This project was designed and developed from Nov 2020 to Feb 2021. I worked on this project as a Product Designer and was responsible for UX design (including delivered solutions and created prototype), visual design, and front-end development.

Platforms & Screen Sizes

Web iPhone and iPadAndroid mobile and tablets

The Problem

According to report from HR team, the bounce rate of the old website was high and the job applying rate was low compared to other job sites. I analyzed the report and summarized into 3 big points:

  • The action button was implicit and too much information in the page which made it hard for user to read and move to next step.
  • The structure of the website was complex and the user-flow was unclear which is confusing to users.
  • The website was old and outdated which didn’t match with company’s core value “Digital Transformation “.

Competitor Analysis

I studied competitors to get familiar with onboarding process. I have studied other job sites and other bank’s recruiting website in Taiwan. There are some insights:

  • Help candidates understand the company’s core value and hiring process is important.
  • Onboarding process needs to be short and easy.
  • To better tailor workouts for users, asking users preferences, goals, etc is helpful.

Design solutions

UX Design

  • Merged information from different window tabs and simplified the content into one single page(including job overview, introduction of company and hiring process ) and focus on job overview by putting it on the top.
  • Reconstructed the user flow to only two hierarchy, which mean there are only two separate tab for the website (Home and Job detail).
  • Use pop-up windows to show job descriptions instead of opening new tab, which save user’s time since they don’t have to move back to previous page when viewing the job description every time.

UI Design

  • Strengthened the brand value by applying brand’s color scheme and use it with bright gradient to provide hi-tech vibe.
  • Use whitespaces to draw the viewer’s eye to the most important area of the page.
  • Add simple interactivity to encourage users to communicate and engage greatly with the website.
  • Attract users by friendly illustrations and motion to show brand’s value "warmth and genuine".

Outcome

The team successfully reveal our new version of official recruitment website. Increased the job applying rate up to 20%. More to do, validated the solutions with usability testing.
